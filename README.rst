# VerificationBot
A Discord Verification Bot for python.
 
Installing
----------

**Python 3.5.3 or higher is required**

.. code:: sh

    # Linux/OS X
    python3 -m pip install -U discord.py

    # Windows
    py -3 -m pip install -U discord.py
